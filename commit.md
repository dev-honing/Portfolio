# 커밋 메시지 유형

| 유형       | 설명                       |
|------------|---------------------------|
| Feat:      | 기능 추가                  |
| Design:    | 스타일링                   | 
| Refactor:  | 리팩토링                   |
| Modify:    | 기능 수정                  |
| Fix:       | 오타 수정                  |
| Move:      | 파일/폴더 구성 이동         |
| Remove:    | 코드 삭제                  |
| Delete:    | 파일/폴더 삭제             |
| Package:   | 패키지 관리                |
| Docs:      | Markdown 문서 작업         |
| Reset:     | 초기화                     |
| Test:      | 테스트 작업                |
| Bad:       | 좋지 않은 코드             |
| Comments:  | 주석 업데이트              |
| Data:      | JSON/DB 업데이트           |
| Chore:     | Prettier 코드 포맷팅       |
